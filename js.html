What is JavaScript?

JavaScript makes websites interactive. It lets buttons do things, changes content, and reacts to user actions.

  • function myFunction() { } — Creates a function (a set of instructions).
  • onclick — Runs code when something is clicked.
  • alert("Hello!") — Shows a popup message.
  • document.getElementById() — Finds something on the page.
  • window.location.href — Sends you to another page.
