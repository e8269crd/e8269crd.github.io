What is CSS?

CSS is what makes websites look nice. It controls colors, fonts, spacing, and layout.

  • body { } — Styles the whole page.
  • color: blue; — Changes the text color.
  • background-color: lightgray; — Changes the background color.
  • font-family: Arial; — Changes the font.
  • padding: 20px; — Adds space inside elements.
  • margin: 10px; — Adds space outside elements.
  • border-radius: 10px; — Rounds the corners.
