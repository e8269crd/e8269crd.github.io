What is HTML?

This page explains the basic building blocks of an HTML webpage.

  • <!DOCTYPE html> — Tells the browser this is HTML.
  • <html> — Wraps the whole webpage.
  • <head> — Info about the page.
  • <title> — Text on the browser tab.
  • <body> — Everything you see.
  • <h1> — Big heading.
  • <p> — Paragraph text.
  • <ul> <li> — Bullet list.
