How to Make a Simple HTML Page

This list shows the basic code you need to make a webpage and what each part does.

  • <!DOCTYPE html> — Tells the computer: “This is an HTML webpage.”
  • <html> (put website code here) </html> — Holds the entire webpage.
  • <head> (page info here) </head> — Stores information about the page that people don’t see.
  • <title> (type page name here) </title> — The name that shows on the browser tab.
  • <body> (stuff people see here) </body> — Everything you can see on the webpage goes here.
  • <h1> (type a big title here) </h1> — Makes a big title on the page.
  • <p> (type your words here) </p> — Makes a paragraph of text.
  • <a href="(put website link here)"> (type link words here) </a> — Makes text you can click to go to another website.
  • <img src="(image file name here)" alt="(say what the picture is)"> — Shows a picture on the page.
  • <ul> <li> (write item here) </li> </ul> — Makes a list with bullet points.
  • <ol> <li> (words here) </li> </ol> — Makes a list with numbers (good for steps).
